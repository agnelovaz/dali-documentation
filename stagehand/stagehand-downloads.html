Ubuntu Linux

To install Stagehand, download the appropriate 64 or 32 bit version.

If you are uncertain which version, type "uname -m" on the command line ("x86_64" is 64 bit, "i686" is 32 bit).

These archives include the required Qt dependencies.

Extract the archive with:

tar zxf stagehand-ubuntu-xxbit-x.xx.tar.gz

cd into the created directory, and run the installer as root:

$ sudo ./install.sh
