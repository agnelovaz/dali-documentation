Installing StageHand

Follow this link for Stagehand binaries for Windows, Linux and OSX.

Connecting Stagehand to your DALi application

Preparing DALi

Stagehand connects to DALi via network using a TCP/IP connection, to enable this, your dali-adaptor RPM must be built with a configure option: --enable-networklogging

To do this, please add this to your current dali-adaptor configure line:

--enable-networklogging

Here is an example dali-adaptor configure line for Ubuntu (change the profile to your desired target device):

$ ./configure --prefix=$DESKTOP_PREFIX --enable-profile=UBUNTU --enable-gles=20 --enable-networklogging

Note: The rest of the build steps should remain the same.

Once this RPM is installed, you can run your DALi application and connect Stagehand to it.

Running the DALi application

To enable network control on DALi, an environment variable must be set:

$ export DALI_NETWORK_CONTROL=1 to enable

Or run the application with the variable set.

$ DALI_NETWORK_CONTROL=1 /usr/apps/com.samsung.dali-demo/bin/dali-demo

Running Stagehand

You can now run Stagehand to connect to DALi and preview the scene by simply running from the command line:

$ stagehand

Note: If you have a proxy, you will need to set your http_proxy / https_proxy environment variables accordingly.

Quick Stagehand configuration

When running for the first time, you will need to tell Stagehand how to connect to DALi. Click the settings icon:

Target Platform: Change to the OS of the device running the DALi application. Target Connection: Set to "Direct" if running Stagehand on the same Ubuntu machine.

If running your DALi application on a different machine / device, change connection to "Forward to device" and setup the Host IP address and port to that of the machine running the DALi application.

Go!

Now you are setup, click the refresh icon (2 circular arrows) to fetch the scene data from the DALi application.

If successful, a wireframe of actors should appear in the center, along with a tree-view hierarchy of actors (by name) and a property window for the currently selected actor.

Troubleshooting

If there are any issues setting up the connection, please see the full Stagehand guide for more troubleshooting instructions.
